# Task 001: Rename create-profile to new-profile

**Phase:** 4 — Command Renames and Cleanup
**Tier:** sonnet-med

## Purpose and scope

Rename the `create-profile` command to `new-profile` across all code and documentation. This is a mechanical rename with no logic changes. The task is small and can be dispatched in parallel with task 002 (help and status).

## Requirements

### File rename: `src/create-profile.sh` → `src/new-profile.sh`

Rename the file. Update the internal function name from `create_profile()` to `new_profile()`. Update all references.

**Inside `src/new-profile.sh`:**
- Rename the function: `function create_profile()` → `function new_profile()`
- Update the file header comment: `# new-profile.sh — generate a profile YAML from local Claude assets.`
- The error message `"Error: unknown option '${arg}' for create-profile"` → `"Error: unknown option '${arg}' for new-profile"`
- The error message `"Error: --name is required for create-profile"` → `"Error: --name is required for new-profile"`

**Inside `src/index.sh`:**
- Change `source ./create-profile.sh` → `source ./new-profile.sh`
- The Phase 1 wiring used `create_profile` as a temporary alias for `new-profile`; update that call to `new_profile`
- Update the dispatch string: any reference to `"create-profile"` → `"new-profile"` (the command name seen by the user)

**Inside `README.md`:**
- Find all occurrences of `create-profile` and replace with `new-profile`
- Update any usage examples showing `ai-sandbox create-profile` → `ai-sandbox new-profile`

### Verify no orphan references

After the rename, search for any remaining `create-profile` or `create_profile` references that should have been updated:

```bash
grep -rn 'create.profile\|create_profile' src/ bin/ test/ README.md
```

Expected survivors: none (or only comments explaining the rename history, if any).

The rollup output `bin/ai-sandbox.sh` is generated — it will be regenerated by `make build`. Do not manually edit it.

## Assumptions

- Phase 1's `index.sh` dispatch wires `new-profile` to call `create_profile` (the old name) as a temporary alias. This task replaces that alias with the real `new_profile()` call.
- No integration tests reference `create-profile` directly (unit tests reference `create_profile()` function name — those will be updated in Phase 5).

## References

- `src/create-profile.sh` → `src/new-profile.sh`
- `src/index.sh` — dispatch string and source line
- `README.md` — user-facing documentation
